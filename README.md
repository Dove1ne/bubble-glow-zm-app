# Bubble Glow ZM App

## Project Structure

- **frontend/**: Contains the React Native Expo app.
- **backend/**: Contains the Node.js/Firebase backend app.
- **.gitignore**: Specifies intentionally untracked files to ignore.
- **README.md**: Provides an overview of the project and setup instructions.
- **package.json**: Dependency manifest for Node.js apps.

## Getting Started

### Prerequisites
- Node.js
- Expo CLI

### Installation Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/Dove1ne/bubble-glow-zm-app.git
   ```
2. Navigate to the `frontend` directory:
   ```bash
   cd frontend
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the Expo project:
   ```bash
   npm start
   ```
5. For the backend, navigate to the `backend` directory and run:
   ```bash
   cd ../backend
   npm install
   ```
6. Start the backend server:
   ```bash
   node index.js
   ```
